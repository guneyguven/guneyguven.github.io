# Images Directory

Place your portfolio images, project screenshots, and blog post images here.

## Recommended Image Organization

- `profile/` - Profile pictures and personal photos
- `projects/` - Screenshots and images for your projects
- `blog/` - Images for blog posts
- `icons/` - Icons and logos

## Image Guidelines

- Use optimized images (compressed for web)
- Recommended formats: JPEG for photos, PNG for graphics with transparency, SVG for logos
- Keep file sizes reasonable (< 500KB per image when possible)
